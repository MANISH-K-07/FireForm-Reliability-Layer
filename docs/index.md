# 🔥 FireForm Reliability Middleware

A safety-preserving middleware layer for structured emergency incident extraction.

FireForm's incident intake pipeline relies on structured outputs generated by Large Language Models (LLMs). However, raw LLM extraction frequently produces:

* Missing schema fields
* Invalid datetime formats
* Free-text severity classifications
* Location structure corruption
* Non-deterministic outputs for identical reports

This project introduces a **Reliability Middleware Layer** that sits between extraction and schema validation to ensure:

* Structural determinism
* Logical consistency
* Safe recoverability
* Zero fabricated operational data

---

## 🔎 What This Prototype Demonstrates

Even when raw extraction fails 96% of the time:

* 61.33% of reports become fully structured and valid
* 90.77% of salvageable reports are safely repaired
* Unsafe guess count remains **0**

The system prioritizes:

> Safety over aggressive auto-completion.
